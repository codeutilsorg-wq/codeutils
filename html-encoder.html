How to Use

  1. To Encode: Enter plain text or HTML in the Input field and click "Encode HTML"
  2. To Decode: Enter HTML entities in the Input field and click "Decode HTML"
  3. Click "Copy Output" to copy the result to clipboard
  4. Click "Clear" to reset both fields

Keyboard Shortcuts

  • Ctrl+Enter - Encode HTML
  • Ctrl+Shift+Enter - Decode HTML

What are HTML Entities?

HTML entities are used to represent special characters in HTML. Some characters are reserved in HTML and must be replaced with character entities to be displayed correctly in a web browser.

Common HTML Entities

  • < (less than) - Encoded as &lt;
  • > (greater than) - Encoded as &gt;
  • & (ampersand) - Encoded as &amp;
  • " (double quote) - Encoded as &quot;
  • ' (apostrophe) - Encoded as &apos;
  • © (copyright) - Encoded as &copy;
  • ® (registered) - Encoded as &reg;
