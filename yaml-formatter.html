About YAML Formatter

YAML (YAML Ain't Markup Language) is a human-readable data serialization format commonly used for configuration files.

YAML Syntax Rules

  • Indentation matters - use consistent spaces (not tabs)
  • Key-value pairs use colons: key: value
  • Lists use dashes: - item
  • Comments start with #
  • Strings don't usually need quotes

Example

server:
  host: localhost
  port: 8080
  features:
    - auth
    - logging
