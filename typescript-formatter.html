About TypeScript Formatter

This tool formats TypeScript code with proper indentation and consistent style.

Features

  • Proper indentation for code blocks
  • Configurable indent size
  • Optional semicolons
  • Single or double quotes
  • Handles interfaces, types, functions, and classes

TypeScript Style Guidelines

  • Use 2 spaces for indentation (common convention)
  • Use semicolons consistently
  • Use PascalCase for types and interfaces
  • Use camelCase for variables and functions

Tips

For full TypeScript formatting, consider using Prettier or ESLint with TypeScript plugin in your development environment.
