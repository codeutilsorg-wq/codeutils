About Line Number Adder

This tool adds or removes line numbers from text for easier reference and documentation.

Options

  • Start at: The starting number (default: 1)
  • Separator: Character(s) between number and text (default: ": ")
  • Pad numbers: Add leading zeros for alignment (1 becomes 01)
  • Skip empty lines: Don't number blank lines

Use Cases

  • Referencing specific lines in code reviews
  • Creating numbered lists from plain text
  • Documentation and tutorials
  • Legal or academic citations
