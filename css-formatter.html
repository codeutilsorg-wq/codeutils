About CSS Formatter

This tool helps you format and beautify CSS code with proper indentation, or minify it for production use. All processing happens in your browser - your code is never sent to any server.

Features

  • Format: Beautify CSS with proper indentation
  • Minify: Remove whitespace and comments to reduce file size
  • Customizable Indent: Choose 2 spaces, 4 spaces, or tabs
  • File Support: Upload CSS files or drag & drop
  • Download: Save formatted CSS as a file

Keyboard Shortcuts

  • Ctrl+Enter: Format CSS
  • Ctrl+M: Minify CSS
  • Ctrl+K: Copy output to clipboard
