About Text Sorter

This tool sorts lines of text using various sorting methods.

Sort Types

  • Alphabetical: Sort A-Z or Z-A
  • Numeric: Sort by number (extracts first number in each line)
  • Length: Sort by line length (shortest to longest or vice versa)
  • Random: Shuffle lines randomly

Options

  • Case insensitive: Treat uppercase and lowercase as equal
  • Remove duplicates: Remove duplicate lines before sorting
  • Trim whitespace: Remove leading/trailing spaces from each line
