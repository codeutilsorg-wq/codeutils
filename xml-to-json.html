About XML to JSON Converter

This tool converts XML documents to JSON format while preserving structure and attributes.

Conversion Rules

  • XML elements become JSON object keys
  • XML attributes are prefixed with @ (e.g., @id)
  • Text content uses #text key when mixed with attributes
  • Repeated elements become arrays

Example

XML:

<person id="1"><name>John</name></person>

JSON:

{"person": {"@id": "1", "name": "John"}}
