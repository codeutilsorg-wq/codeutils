About TSV to JSON

TSV (Tab-Separated Values) is similar to CSV but uses tabs instead of commas as delimiters.

Example

TSV:

name	age
John	30

JSON:

[{"name": "John", "age": 30}]

Tips

  • TSV is often used when data contains commas
  • Copy data directly from Excel/Sheets (it's usually tab-separated)
  • Tabs are represented by the Tab key between columns
