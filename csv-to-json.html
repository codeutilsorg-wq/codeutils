About CSV to JSON Converter

This tool converts CSV (Comma-Separated Values) data into JSON format.

Input Format

name,age,city
John,30,NYC
Jane,25,LA

Output Format

[
  {"name": "John", "age": 30, "city": "NYC"},
  {"name": "Jane", "age": 25, "city": "LA"}
]

Options

  • Delimiter: Character separating values (auto-detect supported)
  • First row is headers: Use first row as object keys
  • Parse numbers: Convert numeric strings to numbers
  • Parse booleans: Convert "true"/"false" to booleans
