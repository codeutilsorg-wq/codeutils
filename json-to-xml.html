About JSON to XML Converter

This tool converts JSON data to XML format.

Conversion Rules

  • JSON object keys become XML element names
  • Arrays create repeated elements with "item" wrapper or key name
  • Primitive values become text content
  • Keys starting with @ are treated as attributes

Example

JSON:

{"person": {"name": "John", "age": 30}}

XML:

<person><name>John</name><age>30</age></person>
