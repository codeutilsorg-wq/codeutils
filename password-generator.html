Security Tips

  • Use at least 12 characters for important accounts
  • Include a mix of character types
  • Use unique passwords for each account
  • Consider using a password manager

Password Strength

  • Weak: Less than 8 characters or limited character set
  • Fair: 8-11 characters with mixed types
  • Strong: 12-15 characters with all types
  • Very Strong: 16+ characters with all types
