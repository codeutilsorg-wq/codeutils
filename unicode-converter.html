About Unicode Converter

This tool converts text to various Unicode escape sequence formats used in different programming languages.

Supported Formats

  • \uXXXX (JavaScript/Java): e.g., \u0048\u0065\u006C\u006C\u006F
  • &#xXXXX; (HTML): e.g., &#x48;&#x65;&#x6C;&#x6C;&#x6F;
  • \XXXX (CSS): e.g., \0048\0065\006C\006C\006F
  • U+XXXX (Code Point): e.g., U+0048 U+0065 U+006C U+006C U+006F

Common Unicode Characters

  • Copyright (©): U+00A9, \u00A9
  • Trademark (™): U+2122, \u2122
  • Euro (€): U+20AC, \u20AC
  • Smiley (☺): U+263A, \u263A
