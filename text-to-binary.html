About Text to Binary

This tool converts text characters to their binary representations and vice versa.

How It Works

Each character in the text is converted to its ASCII/Unicode code point, which is then displayed in binary format.

Examples

  • A = 01000001 (ASCII 65)
  • a = 01100001 (ASCII 97)
  • 1 = 00110001 (ASCII 49)
  • Hello = 01001000 01100101 01101100 01101100 01101111
