About HTML Formatter

This tool helps you format and beautify HTML code with proper indentation, or minify it for production use. All processing happens in your browser - your code is never sent to any server.

Features

  • Format: Beautify HTML with proper indentation
  • Minify: Remove whitespace to reduce file size
  • Customizable Indent: Choose 2 spaces, 4 spaces, or tabs
  • File Support: Upload HTML files or drag & drop
  • Download: Save formatted HTML as a file

Keyboard Shortcuts

  • Ctrl+Enter: Format HTML
  • Ctrl+M: Minify HTML
  • Ctrl+K: Copy output to clipboard
