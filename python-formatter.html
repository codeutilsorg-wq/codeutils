About Python Formatter

This tool formats Python code with proper indentation following PEP 8 guidelines.

Features

  • Fixes indentation for code blocks
  • Configurable indent size (4 spaces is PEP 8 standard)
  • Handles functions, classes, and control structures
  • Optional blank lines between functions

PEP 8 Guidelines

  • Use 4 spaces per indentation level
  • Two blank lines between top-level definitions
  • One blank line between method definitions
  • Maximum line length of 79 characters

Limitations

This is a basic formatter. For full PEP 8 compliance, use tools like Black, autopep8, or YAPF in your development environment.
